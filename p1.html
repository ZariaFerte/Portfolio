PCO software

Software Development for PCO FLIM Camera

During my three-month internship at the University of Technology, Graz in Austria, I developed a software using Python to control a PCO FLIM camera, specialized for studying luminescence. My goal was to replace the useful features of the Nikon's software they were using at the institute.

UX design

  • Analysis of the Nikon software (features, design, user feedback)

  • Interviews : To know and understand the user needs

  • Research about luminescence and the specific camera

Prototyping

Development of early prototypes using Python with only a few features to start with.

Testing and Feedback

Improved the software based on feedback from the scientific team, adding features like data visualization and enhanced controls.
