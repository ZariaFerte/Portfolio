PCO software

Software Development for PCO FLIM Camera

During my three-month internship at the University of Technology, Graz in Austria, I developed a software using Python to control a PCO FLIM camera, specialized for studying luminescence. My goal was to replace the useful features of the Nikon's software they were using at the institute.

UX Design process phases

UX Research

  • Competitive analysis : analyse of the Nikon software (features, design, user feedback)

  • Interviews : To know and understand the user needs

  • Research about luminescence and the specific camera

Prototyping

Developed early prototypes using Python with only a few features to strat with.

Testing and Feedback

Improved the software based on feedback from the scientific team, adding features like data visualization and enhanced controls.

Tools Used

Python : multiple libraries such as Tkinter, Pco, Opencv, Numpy, Matplotlib, Scipy etc.

GitHub Link

View the complete files on my GitHub

Challenges and Learnings

This was my first user interface project. First of all, I learned a lot about luminescence, which was a new field. I also learned how to understand users' needs and translate their requests in order to adapt the interface to their use. And finally, I learned to take a step back from my work, to have a critical eye on what I was doing and to listen to user feedback in order to improve it.
